How to Use Ripple Studio

  • Click, tap, or use a stylus anywhere on the canvas to create layered ripples.
  • Enable drag trails in Settings to paint continuous waves; simultaneous touch points are supported.
  • Use automatic patterns and symmetry for hands-free compositions.
  • Save named presets, mark favorites, or import and export preset JSON files.
  • Use the camera button to save the current canvas as a PNG.
  • Presentation mode hides the interface; press Escape to return.
